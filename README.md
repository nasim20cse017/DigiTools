# 🚀 DigiTools - Premium Digital Tools

* **** ![DigiTools Screenshot](https://i.ibb.co.com/LD5CjMR9/Screenshot-2026-04-10-at-12-03-29-AM.png)


**DigiTools** is a high-performance, modern e-commerce landing page and product management system designed for creators, professionals, and businesses.

It delivers a seamless experience for browsing premium digital assets, managing a shopping cart, and exploring transparent pricing plans — all wrapped in a sleek, responsive UI.

🌐 **Live Demo:** [DigiTools Live Site](https://nasim-sarkar-digitools.netlify.app/)
📂 **Source Code:** [GitHub Repository](https://github.com/nasim20cse017/DigiTools.git)

---

## 🛠️ Technologies Used

Built with a modern frontend ecosystem:

* ⚛️ **React.js** — Component-based architecture for scalable UI
* 🎨 **Tailwind CSS** — Utility-first styling for rapid development
* 🌼 **DaisyUI** — Pre-built, customizable UI components
* 🎯 **Lucide React & React Icons** — Beautiful, scalable icons
* 🔔 **React-Toastify** — Smooth, non-blocking notifications
* 🔤 **Google Fonts (Manrope)** — Clean and modern typography
* ⚡ **Vite** — Fast frontend build tool and development server

## 💻 Local Setup & Installation

Follow these steps to run the project locally:

1. **Clone the repository**

```bash
git clone https://github.com/nasim20cse017/DigiTools.git
```

2. **Navigate to the project folder**

```bash
cd DigiTools
```

3. **Install dependencies**

```bash
npm install
```

4. **Run the development server**

```bash
npm run dev
```

5. **Open in browser**
   The app will usually run at:
   `http://localhost:5173`

6. **Build for production (optional)**

```bash
npm run build
```

7. **Preview the production build (optional)**

```bash
npm run preview
```
