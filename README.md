# 🚀 DigiTools - Premium Digital Tools

DigiTools is a high-performance, modern e-commerce landing page and product management system designed for creators, professionals, and businesses.

It delivers a seamless experience for browsing premium digital assets, managing a shopping cart, and exploring transparent pricing plans — all wrapped in a sleek, responsive UI.

---

## 🛠️ Technologies Used

Built with a modern frontend ecosystem:

- ⚛️ **React.js** — Component-based architecture for scalable UI  
- 🎨 **Tailwind CSS** — Utility-first styling for rapid development  
- 🌼 **DaisyUI** — Pre-built, customizable UI components  
- 🎯 **Lucide React & React Icons** — Beautiful, scalable icons  
- 🔔 **React-Toastify** — Smooth, non-blocking notifications  
- 🔤 **Google Fonts (Manrope)** — Clean and modern typography  

---

## ✨ Key Features

### ⚡ Dynamic Product Gallery
- Responsive 3-column grid layout  
- Loads products dynamically from JSON  
- Includes tags like **Best Seller**, **New**, **Popular**  
- Clean product cards with detailed info  

---

### 🛒 Advanced Cart System
- Centralized state management  
- Add to cart functionality  
- Real-time cart updates in navbar  
- Dedicated checkout view  
- Automatic total price calculation  

---

### 📊 Interactive Pricing & Workflow
- Multi-tier pricing plans  
- Highlighted **Pro Plan**  
- Simple **3-step onboarding guide**  
- Clear and user-friendly layout
